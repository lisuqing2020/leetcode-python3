# [338.比特位计数](https://leetcode-cn.com/problems/counting-bits/)

## 思路
很快就想出来了，知道这个是dp题。

## 代码
```pythonclass Solution:
    def countBits(self, num: int) -> List[int]:
        dp = [0]*(num+1)

        for i in range(1, num+1):
            dp[i] = dp[i//2] + i%2
        
        return dp
```
